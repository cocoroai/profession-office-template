# 士業事務所HPテンプレート

行政書士・税理士・司法書士向けのレスポンシブWebサイトテンプレート

## 技術スタック

- HTML5 / CSS3 / JavaScript
- Google Fonts

## ページ構成

- index.html - TOPページ
- services.html - 業務内容
- pricing.html - 料金案内
- profile.html - 代表プロフィール
- faq.html - よくある質問
- contact.html - お問い合わせ
- reservation.html - 予約ページ

## デプロイ

- ホスティング: GitHub Pages（PUBLICリポジトリ必須）
- URL: https://cocoroai.github.io/profession-office-template/

## 注意

GitHub Pagesを使用しているため、リポジトリはPUBLICのままにすること
